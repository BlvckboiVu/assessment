Theory

  1. The CSS box model works by allowing the web browser supplies each element as a square prism
  2. Class can be used on multiple elements while Id is unique to a single element
  3. Media Query allows you to apply CSS styles depending on a device's media type(such as print vs. screen) or other features or characteristics such as screen resolution or orientation, aspect ratio, browser viewport width or height, user preferences such as preferring reduced motion, data usage transparency.
  4. The margin property controls the space outside an element, while the padding property controls the space inside an element. To create a gap, the padding either grows the element's size or shrink the content inside.
  5. CSS selectors are used to find the HTML elements you want to style. The categories are:
    • Attribute selector
    • Id selector
    • Class selector
    The types of CSS are:
    • external
    • internal
    • inline
