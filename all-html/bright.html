Question 1

How does the CSS box model work?

The CSS box model describes how elements are displayed and laid out on a webpage

Question 2

What is the difference between "class" and "id" in html?

Classes are generally used for styling and applying behaviors to groups of elements. For example, if you want all buttons on a page to have the same style, you would give them all the same class while Ids are used when you need to uniquely identify an element, such as for a single header, a specific section, or when manipulating that element directly via JavaScript.

Question 3

What is Media query?

A media query in CSS is a technique used to apply different styles to a website or web application based on the characteristics of the device or viewport displaying the content.

Question 4

What is the difference between Margin padding and Gap?

Margin creates space outside the element's border, affecting the space between the element and its neighbors. Padding creates space inside the element's border, affecting the space between the content and the border. Gap defines space between items in CSS Grid or Flexbox layouts.

Question 5

What is a css selector? What are the categories and types?

A CSS selector is a pattern used to select and style specific elements in an HTML document. Selectors are used to target elements based on various criteria, allowing you to apply styles to those elements. Categories: 1. Basic selector 2. Combinator selector 3. Attribute Selector 4. Pseudo Selectors
