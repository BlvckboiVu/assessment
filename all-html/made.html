THEORY

The CSS Box Model is essentially a container that wraps around or includes mulitple properties or HTML element. It consists of Content, Padding, Borders and Margin.It is basically a set of rules that dictates how items are displayed within your website or web project.


The difference between Class and Id in HTML; Class can be used on multiple elements, allowing us to style several elements in the same way. On the other hand,"id" is unique and can only be used once per page, making it specific for one element. Basically, class is for multiple elements and id is for a unique element.


A media query is a set of instructions that tells the website how to display based on different factors like screen size, resolution and orientation. It helps to make websites look good on all kinds of devices, from big screens to tiny phones.


Margin is like the space outside the border of an element, creating separation between elements. Padding is the space between the content inside an element and its border. Gap is used in grid layouts to create space beteween grid items.
