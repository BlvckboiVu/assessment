Theory

1. How does the css Box Model Work?

The css Box Model describe the rectangular boxes that are generated for element in the document tree and consist of : mergin,borders,padding, and the actual content.

2. What is the differnt between "class" and "id" in HTML?

A "class" is a reusable style that can be applied to multiple element while an "id" is a unique identifier applied to a single element.

3. What is a media Query?

A Media Query? is a CSS technique used to applu style to different devicesand screen sizes by setting condition for when styles should be applied.

4. What is the difference between Mergin,padding,and Gap

Margin is a space outside an element border, padding is the space inside an element border,and Gap is the space betweenelement in a layout,particularly in a grid and flexbox.

5. What is a CSS selector?

A CSS selector is a pattern used to select selectors(e.g types,class,ID),combinators(e.g...descendant,child)and pseudo-selectors(e.g...,:hover,:nth-child).
