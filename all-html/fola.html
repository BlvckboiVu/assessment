Theory

  1. The CSS Box model defines how different parts of a box (margin, padding, border and content) work together to create a visible box on a webpage
  2. The id attribute is used to uniquely identify a specific element on a web page while the class attribute is used to specify one or more class names for an HTML element.
  3. A media query is a CSS technique that allows you to apply styles to an element based on the characteristics of the device or screen displaying the content.
  4. Margin is the space outside an elements border, Padding is the space between an elements content and its border and Gap is the space between grid or flex items.
  5. A CSS selector is a pattern that specifies which HTML elements should have specific CSS properties applied to them. There are many types and Categories and they are: Simple selectors, Combinator selectors, Pseudo-class selectors, Pseudo-element selectors and Attribute selectors
