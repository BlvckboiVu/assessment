Theory

1. How does the CSS Box Model work?
The CSS Box Model is a fundamental concept that describes the structure of elements on a web page. Each element is represented as a rectangular box consisting of four parts:

  • Content: The actual content of the element (text, images, etc.).
  • Padding: The space between the content and the border. It increases the space around the content inside the border.
  • Border: The line surrounding the padding and content.
  • Margin: The space outside the border, creating distance between the element and other elements.
2. What is the difference between “Class” and “Id” in HTML?
Class: Used to apply styles to multiple elements. Defined with a dot (.) and can be used multiple times on a page. 
<div class="example"></div>
Id: Used to apply styles to a single unique element. Defined with a hash (#) and must be unique within a page. 
<div id="unique"></div>
3. What is a Media Query?
A Media Query is a CSS technique used to apply styles based on the conditions of the device, such as screen size, resolution, orientation, etc. It allows for responsive design by adapting the layout to different devices. 
@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
4. What is the difference between Margin, Padding, and Gap?
  • Margin: The space outside the element's border, separating it from other elements.
  • Padding: The space inside the element's border, around the content.
  • Gap: Used in CSS Grid and Flexbox to define the space between rows and columns within a container.
5. What is a CSS Selector? What are the categories and types?
A CSS Selector is a pattern used to select and style HTML elements. Categories and types include:
  • Simple Selectors: Select elements based on name, id, class, etc.
    • * (Universal)
    • element (Type)
    • .class (Class)
    • #id (ID)
    • [attribute] (Attribute)
  • Combinator Selectors: Select elements based on the relationship between them.
    • ancestor descendant (Descendant)
    • parent > child (Child)
    • previous + next (Adjacent Sibling)
    • previous ~ siblings (General Sibling)
  • Pseudo-class Selectors: Select elements based on their state.
    • :hover, :first-child, etc.
  • Pseudo-element Selectors: Select and style parts of an element.
    • ::before, ::after, etc.
