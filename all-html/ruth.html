question 2

what is the difference between "class"and "id"?

class: this is used to assign one or more class names to an element ID: the id attribute is unique to each of the element on a page. no two element can have the same id

question 3

what is media query

this is a CSS technique used to apply styles based on the characteristics of the device or viewport displaying the content

question 4

what is the difference between margin,padding and gap?

MARGIN: this is the space outside of an element, creating distance that neighboring element. PADDING: is the space inside an element, between the content and the border. GAP: it is used in css grid and flexbox layouts specifying the space between grid, row,column or flex items.

question 5

what is a css selector? what are the categories and types

css selector is a pattern used to select the element you want to style in an HTML Document CATEGORIES: 1. inline CSS 2. internal CSS 3. external css. TYPES:1. Element selector 2. class selectors 3. ID selectors 4. attribute selector.
